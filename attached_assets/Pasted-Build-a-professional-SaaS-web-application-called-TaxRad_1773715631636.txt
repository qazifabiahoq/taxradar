Build a professional SaaS web application called TaxRadar — an AI-powered tax document intelligence platform for CPA firms and tax professionals.

Tagline: Audit-ready in minutes. Built for tax professionals.

Business model: CPA firms and tax professionals upload messy client tax documents. TaxRadar's AI agents scan everything, cross-check income, flag risky deductions, detect missing forms, and produce a CPA-ready tax readiness memo. It does not replace CPAs. It eliminates the 3 to 8 hour manual intake process so they can focus on actual tax strategy.

Backend will be deployed on Render. Frontend calls backend via environment variable VITE_API_URL. All API calls use import.meta.env.VITE_API_URL.

Returns JSON report

GET /health
Returns {"status": "ok"}

CORS: allow all origins
